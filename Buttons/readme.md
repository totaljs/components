## Buttons
Set of button colors, types and sizes.

#### Usage

```html
<div class="button is-small is-mint">Click me</div>
```

#### Sizes
- is-small
- (default)
- is-large

#### Types
- (default)
- is-outlined

#### Colors 
- is-grapefruit
- is-bittersweet
- is-sunflower
- is-grass
- is-mint
- is-aqua
- is-blue-jeans
- is-lavender
- is-pink-rose
- is-light-gray
- is-meidum-gray
- is-dark-gray
