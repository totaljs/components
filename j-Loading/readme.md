## j-Loading

- works as singleton
- __NEW__: supports text `SETTER('loading', 'show', 'YOUR TEXT')`

### Methods

- `SETTER('loading', 'show', [text])` - shows loading
- `SETTER('loading', 'hide', [sleep])` - hides loading
