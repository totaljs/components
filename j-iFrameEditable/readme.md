## j-iFrameEditable

- can edit HTML safely
- height of iframe is automatically counted from the `body`
- __important__: content must contain `contenteditable` attribute

__Configuration__:
- `bind` {String} can contain `body` (default), `html` or `contenteditable`

__Methods__:
- `component.resize()`
