## j-Title

- easy usage
- singleton

__Configuration__:

- `separator {String}` a character for separation (default: `-`)
- `name {String}` a name of the website (default: `document.title`)
- `empty {String}` a string, which is filled, if there is no value in the path

__Methods__:

- `component.rename(value)`
