## j-ObjectTree

Expandable/Collabsible object tree.

__Configuration__:

- `exec` {String} function to call when a user clicks on icon

__Usage__:

```js
function exec_function(type, value) {
    type; // 'path' or 'data'
    value;// copied path(string) or property value(any datatype)
}
```
